Markowitz Portfolio Optimisation Engine

Built a full mean-variance optimisation engine in Python from scratch across a 15-stock, 5-sector universe — the same framework used by institutional asset managers and quant funds as the baseline for portfolio construction.

Expected returns were derived using a 50/50 blend of historical returns and CAPM estimates, with Ledoit-Wolf covariance shrinkage applied to produce more stable estimates. 50,000 Monte Carlo simulations mapped the full opportunity set before SLSQP constrained optimisation identified the exact efficient frontier.

Three constraint scenarios were applied — a 30% single-asset weight cap, a 40% sector cap enforced via inequality constraints, and a 15% short-selling limit — producing six portfolio variants compared across Sharpe ratio, Calmar ratio, and maximum drawdown. The maximum Sharpe portfolio generated 23% expected return at 14% volatility, outperforming an equal-weight benchmark by 8% on a Sharpe Ratio basis.
